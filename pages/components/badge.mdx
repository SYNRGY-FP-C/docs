import Badge from "@components/built-in/Badge";

## Badge

| Props    | Type              | Required | Default   |
| -------- | ----------------- | -------- | --------- |
| variant  | string, enum      | false    | `primary` |
| outline  | boolean           | false    | `false`   |
| method   | string, enum      | false    | `null`    |
| children | string, ReactNode | true     | `Badge`   |

Variants:

- `primary` (default)
- `success`
- `info`
- `warning`
- `danger`

Methods:

- `POST`
- `GET`
- `PUT`
- `DELETE`
