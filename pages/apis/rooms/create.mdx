## Create Room

**Request :**

- Method : <Badge method="POST">POST</Badge>
- Endpoint : `/api/rooms`
- Authorization: `ADMIN only`
- Headers :
  - `Authorization: Bearer accessToken`
  - `Content-Type: application/json`
  - `Accept: application/json`

**Body :**

```json
{
  "title": "string",
  "description": "string",
  "type": "string|enum",
  "payment_period": "string|enum",
  "images": ["string"],
  "location": "string",
  "facilities": ["string"]
}
```

**Response :** <Badge outline variant="success">201 Created</Badge>
