## User Registration `MVP`

**Request :**

- Method : <Badge method="POST">POST</Badge>
- Endpoint : `/api/users/register`
- Headers :
  - `Content-Type: application/json`
  - `Accept: application/json`

**Body :**

```json
{
  "fullname": "string",
  "email": "string",
  "password": "string"
}
```

**Response :** <Badge outline variant="success">201 Created</Badge>

```json
{
  "code": "number",
  "status": "string, enum",
  "data": {
    "accessToken": "string"
  }
}
```
