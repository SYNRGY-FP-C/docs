## User Login `MVP`

**Request :**

- Method : <Badge method="POST">POST</Badge>
- Endpoint : `/api/users/login`
- Headers :
  - `Content-Type: application/json`
  - `Accept: application/json`

**Body :**

```json
{
  "email": "string",
  "password": "string"
}
```

**Response :** <Badge outline variant="success">200 OK</Badge>

```json
{
  "code": "number",
  "status": "string, enum",
  "data": {
    "accessToken": "string"
  }
}
```
