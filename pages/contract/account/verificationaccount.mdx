## Change Password

**Request :**

- Method : <Badge method="POST">POST</Badge>
- URL : `https://be-java-binar-fp-staging.up.railway.app`
- Endpoint : `/v1/account/verification`
- Headers :
  - `Authorization: Bearer <accessToken>`
  - `Content-Type: multipart/form-data`

**Body :**

```json
{
  "email": "string",
  "phone": "string",
  "type": "string",
  "photo": "file, png|jpg"
}
```

