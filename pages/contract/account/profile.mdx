## Update Profile `MVP`

**Request :**

- Method : <Badge method="POST">POST</Badge>
- Endpoint : `/v1/account/profile`
- Headers :
  - `Authorization: Bearer <accessToken>`
  - `Content-Type: multipart/form-data`

**Body :**

```json
{
  "fullname": "string",
  "birthdate": "string, dd-mm-yyyy",
  "gender": "string, enum",
  "occupation": "string",
  "photo": "file"
}
```

**Response :** <Badge outline variant="success">200 OK</Badge>

```json
{
  "status": "string",
  "message": "string"
}
```
