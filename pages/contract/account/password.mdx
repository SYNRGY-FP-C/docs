## Change Password `MVP`

**Request :**

- Method : <Badge method="POST">POST</Badge>
- Endpoint : `/v1/account/password`
- Headers :
  - `Authorization: Bearer <accessToken>`
  - `Content-Type: application/json`

**Body :**

```json
{
  "password": "string",
  "new_password": "string"
}
```

**Response :** <Badge outline variant="success">200 OK</Badge>

```json
{
  "status": "string",
  "message": "string"
}
```
