## User Registration - Penyewa `MVP`

**Request :**

- Method : <Badge method="POST">POST</Badge>
- Endpoint : `/v1/auth/penyewa/register`
- Headers :
  - `Content-Type: application/json`
  - `Accept: application/json`

**Body :**

```json
{
  "email": "string",
  "phone": "string",
  "password": "string"
}
```

**Response :** <Badge outline variant="success">201 Created</Badge>

```json
{
  "status": "string",
  "message": "string",
  "data": {
    "accessToken": "string, jwt",
    "user_details": {
      "id": "number",
      "fullname": "string",

      "occupation": "string",

        "id": "number",

      },

}
```
