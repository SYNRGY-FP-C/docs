## User Registration `MVP`

**Request :**

- Method : <Badge method="POST">POST</Badge>
- Endpoint : `/api/users/register` or else
- Headers :
  - `Content-Type: application/json`
  - `Accept: application/json`

**Body :**

Pencari

```json
{
  "email": "string",
  "phone_number": "string",
  "password": "string",
  "confirm_password": "string",
  "type": 1
}
```

```json
{
  "email": "string",
  "phone_number": "string",

  "type": 2

```
