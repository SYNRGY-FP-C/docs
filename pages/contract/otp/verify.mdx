## OTP Verification

User who already verified their account will get error response.

**Request :**

- Method : <Badge method="POST">POST</Badge>
- URL : `https://backend-synrgy-fp-c.zekhoi.dev`
- Endpoint : `/v1/verify`
- Headers :
  - `Content-Type: application/json`
  - `Accept: application/json`

**Body :**

Bisa menerima `email` atau `phone_number` (harus salah satu)

```json
{
  "email?": "string",
  "phone_number?": "string",
  "code": "string"
}
```

**Response :** <Badge outline variant="success">200 OK</Badge>

```json
{
  "status": "success",
  "message": "Verification succeed"
}
```
