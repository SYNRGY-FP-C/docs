## Add new Kost `MVP`

**Request :**

- Method : <Badge method="POST">POST</Badge>
- URL : `https://be-java-binar-fp-staging.up.railway.app`
- Endpoint : `/v1/kost/addkost` or else
- Headers :
  - `Authorization: Bearer <accessToken>`
  - `Content-Type: multipart/form-data`

**Body :**

```json
{
  "name": "string",
  "outdoor_photo": "file, png|jpg|jpeg",
  "indoor_photo": "file, png|jpg|jpeg",
  "type": "string, enum",
  "description": "string",
  "longitude": "string",
  "latitude": "string",
  "address": "string",
  "province": "string",
  "city": "string",
  "district": "string",
  "adress_note": "string",
  "additional_rule": "string",
  "payment_scheme": [
    {
      "id": "number"
    }
  ],
  "rules": [
    {
      "id": "number"
    }
  ]
}
```

**Response :** <Badge outline variant="success">201 Created</Badge>

```json
{
  "status": "string",
  "message": "string"
}
```
