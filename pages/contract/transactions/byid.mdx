## Get Transaction By ID `MVP`

**Request :**

- Method : <Badge method="GET">GET</Badge>
- URL : `https://be-java-binar-fp-staging.up.railway.app`
- Endpoint : `/v1/transactions/history/:id` or else
- Headers :
  - `Authorization: Bearer <access_token>`
  - `Content-Type: application/json`
  - `Accept: application/json`

**Response :** <Badge outline variant="success">200 OK</Badge>

```json
{
  "status": "string",
  "message": "string",
  "data": [
    {
      "id": "number",
      "tf_end_date": "date",
      "price": "number"
    }
  ]
}
```
