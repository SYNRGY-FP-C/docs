## Get Transaction Owner `MVP`

**Request :**

- Method : <Badge method="PUT">PUT</Badge>
- URL : `https://be-java-binar-fp-staging.up.railway.app`
- Endpoint : `/v1/transactions/owner/:id` or else
- Headers :
  - `Authorization: Bearer <access_token>`
  - `Content-Type: application/json`
  - `Accept: application/json`

**Body :**

```json
{
  "status": "string, onproccess|canceled|approved|rejected|ongoing|ended"
}
```

**Response :** <Badge outline variant="success">200 OK</Badge>

```json
{
  "status": "string",
  "message": "string",
  "data": "object",
  "error": "string"
}
```
