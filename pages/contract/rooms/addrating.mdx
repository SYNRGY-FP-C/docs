## Rating a room / kamar

**Request :**

- Method : <Badge method="POST">POST</Badge>
- URL : `https://be-java-binar-fp-staging.up.railway.app`
- Endpoint : `/v1/rooms/rating` or else
- Headers :
  - `Authorization: Bearer <access_token>`
  - `Content-Type: application/json`
  - `Accept: application/json`

**Body :**

```json
{
  "room_id": "number",
  "rating": "number",
  "review": "string",
  "anonym": "boolean"
}
```

**Response :** <Badge outline variant="success">201 Created</Badge>
