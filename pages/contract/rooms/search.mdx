## Search Rooms `MVP`

**Request :**

- Method : <Badge method="GET">GET</Badge>
- URL : `https://be-java-binar-fp-staging.up.railway.app`
- Endpoint : `/v1/public/search` or else
- Headers :
  - `Content-Type: application/json`
  - `Accept: application/json`
- Query Parameters :
  - `keyword` : `string`
  - `label` : `string`
  - `type`: `string`
  - `price_min`: `number`
  - `price_max`: `number`
  - `size`: `number`
