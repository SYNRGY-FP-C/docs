## Add new Room `MVP`

**Request :**

- Method : <Badge method="POST">POST</Badge>
- URL : `https://be-java-binar-fp-staging.up.railway.app`
- Endpoint : `/v1/rooms/add` or else
- Headers :
  - `Authorization: Bearer <access_token>`
  - `Content-Type: application/json`
  - `Accept: application/json`

**Body :**

```json
{
  "name": "string",
  "images": ["string, base64"],
  "quantity": "number",
  "price": "number",
  "length": "number",
  "width": "number",
  "max_person": "number",
  "indoor_bathroom": "boolean",
  "bathroom_facilities": [
    {
      "id": "number"
    }
  ],
  "bedroom_facilities": [
    {
      "id": "number"
    }
  ],
  "addons_facilities": [
    {
      "name": "string",
      "price": "number"
    }
  ]
}
```

**Response :** <Badge outline variant="success">201 Created</Badge>

```json
{
  "status": "string",
  "message": "string"
}
```
