## Contribute

1. Clone repository

```bash
git clone -b <branchname>  <remote-repo-url>
```

```bash filename="Using SSH"
git clone -b development git@github.com:SYNRGY-FP-C/docs.git
```

```bash filename="Using HTTPS"
git clone -b development https://github.com/SYNRGY-FP-C/docs.git
```

2. Checkout branch feature/namafitur

```bash
git checkout -b feature/namafitur
```

3. Pull dari branch development

```bash
git pull origin development
```

4. Add, Commit dan Push ke branch feature/namafitur

```bash
git add .
```

```bash
git commit -m "commit message"
```

```bash
git push origin feature/namafitur
```

5. Pull Request ke branch development di github
