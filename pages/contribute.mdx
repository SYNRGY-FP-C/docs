## Contribute

### 1. Clone repository

```bash
git clone -b <branchname> <remote-repo-url>
```

```bash filename="Example Using SSH"
git clone -b development git@github.com:SYNRGY-FP-C/frontend-final-project.git
git clone -b development git@github.com:SYNRGY-FP-C/backend-final-project.git
```

```bash filename="Example Using HTTPS"
git clone -b development https://github.com/SYNRGY-FP-C/frontend-final-project.git
git clone -b development https://github.com/SYNRGY-FP-C/backend-final-project.git
```

### 2. Checkout branch feature/namafitur-step

```bash
git checkout -b feature/namafitur-step
```

```bash filename="Example"
git checkout -b feature/login-slicing
```
