## Model

Model sequelize berfungsi untuk mengatur struktur database yang akan digunakan. Model ini juga berfungsi untuk mengatur relasi antar tabel. Model ini juga berfungsi untuk mengatur validasi data yang akan dimasukkan ke dalam database.

### Example

```js
const Sequelize = require("sequelize");

module.exports = (sequelize) => {
  const User = sequelize.define("user", {
    id: {
      type: Sequelize.INTEGER,
      autoIncrement: true,
      primaryKey: true,
    },
    name: {
      type: Sequelize.STRING,
      allowNull: false,
    },
    email: {
      type: Sequelize.STRING,
      allowNull: false,
      unique: true,
    },
    password: {
      type: Sequelize.STRING,
      allowNull: false,
    },
  });

  return User;
};
```
