## Mock API

Mock API adalah sebuah API yang dibuat untuk memudahkan pengembangan aplikasi. Biasanya API ini dibuat dengan menggunakan data-data yang sudah ada. Contohnya, kita memiliki sebuah aplikasi yang membutuhkan data user. Kita bisa membuat API yang mengembalikan data-data user yang sudah ada. Dengan begitu, kita bisa mengembangkan aplikasi tanpa harus menunggu API yang sebenarnya sudah siap.

### Example

Membuat api dengan fakerjs

```javascript
const express = require("express");

const app = express();

const faker = require("faker");

app.get("/api/users", (req, res) => {
  const users = [];
  for (let i = 0; i < 10; i++) {
    users.push({
      id: faker.random.uuid(),
      name: faker.name.findName(),
      email: faker.internet.email(),
      avatar: faker.internet.avatar(),
    });
  }
  res.json(users);
});

app.listen(3000, () => {
  console.log("Server is running on port 3000");
});
```
