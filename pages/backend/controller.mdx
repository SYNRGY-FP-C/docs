## Controller

Controller berfungsi untuk mengatur dan mengendalikan semua proses yang terjadi pada sistem. Controller menerima request dari user dan mengirimkan request tersebut ke model. Controller juga menerima response dari model dan mengirimkan response tersebut ke view. Controller juga dapat mengatur tampilan yang akan ditampilkan ke user.

## Example

```js filename="example.controller.js"
const createExample = async (req, res, next) => {
  try {
    const { name, description } = req.body;
    res.status(201).json({
      code: 201,
      message: "Example created successfully",
      data: {
        name,
        description,
      },
    });
  } catch (error) {
    next(error);
  }
};

const getExample = async (req, res, next) => {
  try {
    const { id } = req.params;
    res.status(200).json({
      code: 200,
      message: "Example retrieved successfully",
      data: {
        id,
      },
    });
  } catch (error) {
    next(error);
  }
};

const updateExample = async (req, res, next) => {
  try {
    const { id } = req.params;
    const { name, description } = req.body;
    res.status(200).json({
      code: 200,
      message: "Example updated successfully",
      data: {
        id,
        name,
        description,
      },
    });
  } catch (error) {
    next(error);
  }
};

const deleteExample = async (req, res, next) => {
  try {
    const { id } = req.params;
    res.status(200).json({
      code: 200,
      message: "Example deleted successfully",
      data: {
        id,
      },
    });
  } catch (error) {
    next(error);
  }
};
```
