## JWT

JWT adalah singkatan dari JSON Web Token. JWT adalah sebuah token yang digunakan untuk mengotentikasi sebuah user.

### Example

```js
const { generateToken, verifyToken } = require("@utils/jwt");

const user = {
  id: 1,
  name: "John Doe",
  email: "email@email.com",
};

const token = generateToken(user);

const decoded = verifyToken(token);
```
