## Profil Saya Page

### Description

This page is used to show user's profile and edit user's profile

- Title => `Profil Saya`
- URL => `/my`
- Path => `src/pages/my/index.jsx`

### Authorization / Access

- Only user (tenant / renter) can access this page
- User (tenant / renter) must be logged in to access this page
- User (tenant / renter) can only edit his/her own profile
- If user (tenant / renter) is not logged in, redirect to login page `/login`

### APIs / Services / Contexts

- Is user logged in => `isAuthenticated` using `AuthContext`
- Get current profile => `user` using `AuthContext`

### Forms

- Form Image
  - `avatar`
  - Button => `Unggah`
  - Button => `Hapus`
- Form Profile
  - `fullname`
  - `birthdate`
  - `email`
  - `phone_number`
  - `gender`
  - `occupation`
  - `ktp`
  - Button => `Simpan`
  - Button => `Reset`

### Page Elements / Components / Layouts

- Navbar (using DefaultLayout)
- Form Image
- Form Profile
- Footer (using DefaultLayout)
