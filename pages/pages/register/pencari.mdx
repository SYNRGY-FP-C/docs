## Register Pencari Page

### Description

This page is used to register a new user.

- Title => `Buat akun - Pencari`
- URL => `/register/pencari`
- Path => `src/pages/register/pencari.jsx`

### Authorization / Access

- Everyone can access this page
- If user is already logged in, redirect to `/`

### APIs / Services / Contexts

- Register user => `registerUser` or `POST /api/users/register/pencari` using `AuthContext`

### Forms

- Register Form
  - `fullname`
  - `phone_number`
  - `email`
  - `password`
  - Button => `Daftar`

### Page Elements / Components / Layouts

- Navbar (using DefaultLayout)
- Register Form
- Image
- Footer (using DefaultLayout)
