## Admin History Transactions Page

### Description

This page is used to show user (provider / owner) history transactions (rental, payment, etc) in a table.

- Title => `Dashboard - History`
- URL => `/admin/history`
- Path => `src/pages/admin/history.jsx`

### Authorization / Access

- Only user (provider / owner) can access this page
- User (provider / owner) must be logged in to access this page
- If user (provider / owner) is not logged in, redirect to login page `/login`

### APIs / Services / Contexts

- Is user logged in => `isAuthenticated`, `isAdmin` using `AuthContext`
- Get user (tenant / renter) history transactions => `GET /api/users/transactions` using `transactionService`
