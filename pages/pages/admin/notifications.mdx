## Admin Notification Page

### Description

This page is used to display all notifications for user (provider / owner).

- Title => `Dashboard - Notification`
- URL => `/admin/notifications`
- Path => `src/pages/admin/notifications.jsx`

### Authorization / Access

- Only user (provider / owner) can access this page
- User (provider / owner) must be logged in to access this page
- If user (provider / owner) is not logged in, redirect to login page `/login`

### APIs / Services / Contexts

- Is user logged in => `isAuthenticated`, `isAdmin` using `AuthContext`
- Get user (tenant / renter) notification => `GET /api/notifications` using `notificationService`
